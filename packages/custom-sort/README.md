# Iwara Custom Sort

Userscript that automatically sorts teaser images on iwara.tv (`/videos`,
`/images`, `/subscriptions`, `/users`, `/playlist`, and sidebars) using a
customizable sort function, and can load and sort multiple pages at once.

iwara.tvのサムネイルを自動的にソートするユーザースクリプトです。ソート方法は
カスタマイズでき、一度に複数のページを読み込んでソートできます。

Published on Greasy Fork. Internal versions are tracked in
[`CHANGELOG.md`](CHANGELOG.md).

## Development

```sh
pnpm dev      # dev server with live rebuild
pnpm build    # userscript at dist/main/index.user.js
```

Builds bundle with webpack; large runtime dependencies are loaded as pinned
CDN `@require` headers generated by `@userscript/cdn-from-dependency`. The
userscript metadata block lives in `dist/main/index.meta.js` alongside the
bundle. Whole-program type checking runs inside every build via
fork-ts-checker.
