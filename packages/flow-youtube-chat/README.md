# Flow Youtube Chat

Userscript that flows YouTube live-chat messages across the video,
niconico/danmaku-style. Runs on `https://www.youtube.com/*`.

Youtubeのチャットをニコニコ風に画面上へ流すユーザースクリプトです。

Published on Greasy Fork; user-facing changelogs live in
[`CHANGELOG_EN.md`](CHANGELOG_EN.md) and [`CHANGELOG_JP.md`](CHANGELOG_JP.md)
(the changesets [`CHANGELOG.md`](CHANGELOG.md) tracks internal versions).

## Features

- Flowing chat overlay with configurable appearance (font, size, speed,
  opacity, shadow) via an in-page settings panel
- Chat filtering, including banned words/users (a typed filter-expression
  rework is in progress — see [`docs/backlog.md`](../../docs/backlog.md))
- Chat-field simplification and other per-setting tweaks, with settings
  synced across tabs

## Development

```sh
pnpm dev      # dev server with live rebuild
pnpm build    # userscript at dist/main/index.user.js (prettified)
```

Builds bundle with webpack; large runtime dependencies are not bundled but
loaded as pinned CDN `@require` headers generated by
`@userscript/cdn-from-dependency`. Whole-program type checking runs inside
every build via fork-ts-checker. Work-in-progress sources are excluded in
[`tsconfig.exclude.json`](tsconfig.exclude.json). Feature backlog:
[`docs/backlog.md`](../../docs/backlog.md).
